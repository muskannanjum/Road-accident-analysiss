# Road-accident-analysiss
This dashboard provides an interactive analysis of road accident data to help understand accident patterns and trends. It highlights key insights such as the number of accidents, locations with high accident rates, types of vehicles involved, and time-based accident distribution. 
The Road Accident Analysis Dashboard aims to provide insights into accident trends, hotspots, and contributing factors to inform data-driven decisions for road safety improvement and policy-making.
power bi ,sql,Data modeling
Source : Microsoft Power bi , google
- Accident Hotspots: Interactive map view to identify high-risk locations
- Trend Analysis: Visualize accident trends over time (monthly, yearly)
- Cause-wise Breakdown: Analyze accidents by cause (speeding, drunk driving, etc.)
- Severity Analysis: View accidents by severity (fatal, injury, property damage)
- Demographic Insights: Analyze accidents by age, gender, vehicle type
- Time-wise Analysis: Identify peak accident hours, days, months
- Location Filters: Filter data by state, city, district, or specific road
